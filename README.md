🍽️ Restaurant Reservation Demo
📖 Overview
This project is a restaurant reservation interface inspired by luxury dining websites such as The French Laundry and platforms like Tock.
It demonstrates how HTML, CSS, and JavaScript can be combined to build a clean, responsive, and interactive booking system.


🛠️ Technologies Used
- HTML5 → Structure of the page.
- CSS3 → Styling, layout, and responsive design.
- JavaScript (ES6) → Event handling, form validation, and dynamic updates.
